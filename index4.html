HTML Attributes

  • All HTML elements can have attributes

  • Attributes provide additional information about elements

  • Attributes are always specified in the start tag

  • Attributes usually come in name/value pairs like: name="value"

    The src Attribute

    The < img > tag is used to embed an image in an HTML page. The src attribute specifies the path to the image to be displayed:

  • Absolute URL - Links to an external image that is hosted on another website. Example: src="https://www.w3schools.com/images/img_girl.jpg". Notes: External images might be under copyright. If you do not get permission to use it, you may be in violation of copyright laws. In addition, you cannot control external images; it can suddenly be removed or changed.
